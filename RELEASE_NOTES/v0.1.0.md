# FounderBooster v0.1.0 Release Notes

Summary:
Initial public OSS release of the FounderBooster CLI for publishing local apps via Cloudflare.

Highlights:
- Auto mode (Docker-first) bootstrap.
- Manual mode (port-first) bootstrap.
- Cloudflare tunnel + DNS automation with per-app/env local state.

Install / upgrade:
- Build from source or use `install.sh`.

Links:
- Repo: https://github.com/founderbooster/founderbooster
- Docs: https://github.com/founderbooster/founderbooster/tree/main/docs
