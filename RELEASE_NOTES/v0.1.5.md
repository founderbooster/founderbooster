# FounderBooster v0.1.5 Release Notes

Summary:
This release expands Linux support and improves tunnel safety and update flows.

Highlights:
- Linux (amd64/arm64) prebuilt tarballs and installer updates.
- Dev channel support for install/self-update testing.
- Hashed tunnel names to avoid unintended multi-connector load balancing.

Install / upgrade:
- `fb self update` (Early Access) or reinstall from `install.sh`.
