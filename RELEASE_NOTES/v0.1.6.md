# FounderBooster v0.1.6 Release Notes

Summary:
This release stabilizes lifecycle behavior and improves automation hooks. It adds first-party plugin management for
optional Early Access plugins while keeping core workflows fully open source.

Highlights:
- Plugin install/update/list/remove commands with checksum verification and version gating.
- `fb self update --with-plugins` for combined core + plugin updates.
- Structured `FB_*` machine output lines after `fb bootstrap`.
- Expanded unit/integration/E2E test harnesses and lifecycle messaging improvements.

Install / upgrade:
- `fb self update` (Early Access) or reinstall from `install.sh`.
